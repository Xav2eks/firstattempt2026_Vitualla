## 🚀 Installation & Setup

Follow these steps carefully to replicate and run this project on a different computer.

### Prerequisites

Make sure the following are installed on your machine before proceeding:

| Tool | Version | Download |
|------|---------|----------|
| Node.js | v20 to avoid issue | https://nodejs.org |
| NVM | Latest | https://www.nvmnode.com/guide/download.html |
| Meteor.js | Latest | https://www.meteor.com/install |
| Git | Latest | https://git-scm.com |

---

### Step 0 — Install node 20 using nvm in terminal

```bash
nvm install 20
nvm use 20
```

Verify installation:
```bash
node -v
```

- It should output something like v20.x.x.

---

### Step 1 — Clone the Repository

Open your terminal and run:

```bash
git clone https://github.com/Xav2eks/firstattempt2026_Vitualla.git
cd firstattempt2026_Vitualla
```

---

### Step 2 — Install Meteor (if not installed)

**For macOS / Linux:**
```bash
curl https://install.meteor.com/ | sh
```

**For Windows:**  

Installation in terminal:
```bash
npx meteor
```

Verify installation:
```bash
meteor --version
```

---

### Step 3 — Install Node Dependencies

```bash
meteor npm install
```

> If you encounter permission errors on macOS/Linux, use `sudo npm install`.

---

### Step 4 — Run the Application

```bash
meteor run
```

The app will be available at:
http://localhost:3000
